# Dataset

This project uses the **Individual Household Electric Power Consumption** dataset from the UCI Machine Learning Repository.

## Download

https://archive.ics.uci.edu/ml/datasets/individual+household+electric+power+consumption

## Instructions

1. Download the dataset.
2. Extract the archive.
3. Place the file:

```
household_power_consumption.txt
```

inside this `datasets/` folder.

The dataset is not included in this repository because it exceeds GitHub's file size limit.
