# EcoWatt AI

AI-powered Household Electricity Forecasting and Energy Optimization System.

## Features

- 7-Day Energy Consumption Forecasting
- Monthly Bill Estimation
- Peak Usage Analysis
- Energy Health Score
- Personalized Energy Saving Recommendations
- Interactive Analytics Dashboard

## Tech Stack

- Python
- Pandas
- NumPy
- Prophet
- Streamlit
- Matplotlib
- Plotly
- Scikit-learn

## Dataset

This project uses the UCI Individual Household Electric Power Consumption dataset.

The dataset is not included in this repository due to GitHub file size limits. See `datasets/README.md` for download instructions.
